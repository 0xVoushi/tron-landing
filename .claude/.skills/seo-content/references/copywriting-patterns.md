# Copywriting Patterns Reference

Proven copywriting frameworks for landing page content. Absorbs the core of the `copywriting` marketing skill.

## Headline Formulas

### For Hero Sections (H1)
1. **[Outcome] for [Audience]**
   - "Production-Grade Software for Ambitious Teams"
2. **We [Verb] [What] That [Benefit]**
   - "We Build Products That Ship Fast and Scale"
3. **[Adjective] [Service]. [Proof/Promise].**
   - "Expert Development. 40+ Products Shipped."
4. **Your [Thing] Deserves [Outcome]**
   - "Your Product Deserves Engineers Who Care"
5. **From [Pain] to [Gain]**
   - "From Idea to Production in Weeks, Not Months"

### For Section Headers (H2)
1. **How We [Process/Benefit]**
   - "How We Build" / "How We Work"
2. **[Service Category] That [Outcome]**
   - "AI Integration That Actually Ships"
3. **Why [Audience] Choose [Us/This]**
   - "Why Startups Choose [Studio Name]"
4. **Everything You Need to [Goal]**
   - "Everything You Need to Launch"

## Persuasion Frameworks

### AIDA (Attention → Interest → Desire → Action)
Best for: Full page flow, email sequences

```
Attention: Bold headline addressing a key pain
Interest: Expand on the problem, show understanding
Desire: Present the solution, paint the outcome
Action: Clear CTA with reduced friction
```

### PAS (Problem → Agitate → Solution)
Best for: Hero sections, service descriptions

```
Problem: "Building software is expensive and slow."
Agitate: "Most agencies charge $200/hr and still miss deadlines."
Solution: "We deliver production-grade code on fixed timelines."
```

### FAB (Feature → Advantage → Benefit)
Best for: Service cards, feature lists

```
Feature: "Next.js 15 with static site generation"
Advantage: "Pages load instantly, no server processing needed"
Benefit: "Your visitors stay longer, Google ranks you higher"
```

### BAB (Before → After → Bridge)
Best for: Case studies, testimonials section intros

```
Before: "Their checkout flow had 60% abandonment"
After: "After our rebuild: 12% abandonment, 3x revenue"
Bridge: "Here's how we did it..."
```

## CTA Best Practices

### Primary CTA (Main conversion)
- **Action verb + Value**: "Start Your Project", "Book a Discovery Call"
- **Reduce anxiety**: "Free Consultation", "No Commitment"
- **Be specific**: "Get a Technical Assessment" > "Contact Us"

### Secondary CTA (Lower commitment)
- **Exploration**: "See Our Work", "View Case Studies"
- **Learning**: "How We Work", "Read Our Process"
- **Evidence**: "Meet the Team", "Read Client Stories"

### CTA Placement
- Hero section: Primary + Secondary
- After services section: Primary
- After social proof: Primary
- FAQ section: Primary (objections just answered)
- Footer: Primary (last chance)

### CTA Don'ts
- "Submit" (generic, no value)
- "Click Here" (inaccessible, vague)
- "Buy Now" (wrong for a service business)
- Multiple competing CTAs in the same viewport

## Content Structure Patterns

### The Inverted Pyramid
Lead with the most important information. Each paragraph can be the last one read.

```
Most important → Summary/headline
Important → Key benefits, differentiation
Supporting → Details, methodology, tech stack
Nice to have → Background, history, extras
```

### Scannable Content
- Bullet points for 3+ items
- Bold key phrases (one per paragraph)
- Short paragraphs (2-3 sentences max)
- Subheadings every 3-4 paragraphs
- Numbers and statistics stand out

### Social Proof Hierarchy (Strongest → Weakest)
1. Revenue/growth metrics: "Helped generate $2M in first year"
2. Named client logos: "Trusted by [Company], [Company], [Company]"
3. Specific testimonials with name, role, company
4. Aggregate stats: "40+ products shipped"
5. Generic testimonials without attribution
6. Self-claimed awards or rankings

## Landing Page Section Copywriting

### Hero
- H1: 6-12 words, primary keyword, clear value proposition
- Subhead: 15-25 words, expand on H1, address audience
- CTA: 3-5 words, action + value

### Services
- H2: Service category name (keyword-rich)
- Body: 2-3 sentences per service, outcome-focused
- Each service answers: "What do you get? Why does it matter?"

### Process / How We Work
- H2: "How We Work" or "Our Process"
- 3-5 steps, numbered or sequenced
- Each step: short title + 1-2 sentence description
- Purpose: reduce anxiety, set expectations

### FAQ
- 5-8 questions minimum
- Questions should be actual search queries (long-tail keywords)
- Answers: 40-60 words (ideal for featured snippets)
- Address objections: pricing, timeline, process, guarantees

### Final CTA
- H2: Warm, action-oriented ("Ready to Build Something?")
- Subtext: 1-2 sentences, reduce friction, restate value
- Button: Primary CTA (same as hero for consistency)
