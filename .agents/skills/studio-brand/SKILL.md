# Studio Brand Skill

## Overview

Central brand identity repository for the development studio. Houses naming decisions, brand values, visual identity, and content voice guidelines. All content and design work should reference this skill for consistency.

**Status**: In progress — the studio name and visual identity are not yet finalized. Use `/seo-content-start` with naming exploration enabled to drive these decisions.

## When to Use This Skill

- Writing any copy (headlines, descriptions, CTAs, meta tags)
- Choosing colors, typography, or visual direction
- Making positioning decisions (how to differentiate from competitors)
- Preparing content for a new section or page
- Reviewing content for brand consistency

## Brand Foundation

### Studio Focus (Confirmed)
Full-spectrum development studio offering:
1. **Web Development** — marketing sites, web apps, SPAs, landing pages
2. **Fullstack Product Development** — end-to-end from idea to deploy
3. **AI Integration** — LLM applications, AI-powered features, automation
4. **Web3 / Blockchain** — smart contracts, DeFi, dApps, token infrastructure
5. **Custom Development** — bespoke solutions across diverse tech stacks

### Target Audience (Confirmed)
Mixed — from startups to enterprise:
- Startup founders needing MVPs or technical co-founders
- SMB owners needing web presence or custom tools
- Enterprise teams needing specialized dev capacity (AI, Web3)
- CTOs evaluating agency capabilities
- Crypto projects needing smart contract / DeFi infrastructure

### Positioning (To Be Refined)
Key differentiators to explore:
- Technical depth across multiple cutting-edge domains (AI + Web3 is rare)
- Full ownership — design to deploy, no hand-offs
- Performance obsession — everything ships fast and loads fast
- Small team, senior engineers — no juniors learning on your project
